overseer
========
``node`` app for monitoring and controlling. Current version includes:
* ups parameters monitor;
* ``announcer`` layout

installation
------------
``git clone https://github.com/longdeer/overseer.git``

configuration
-------------
.env
```
APP_NAME=
LISTEN_ADDRESS=
LISTEN_PORT=
LOGGY_FILE=
UPS_ADDRESSES=
UPS_NAMES=
UPS_SNMP_POLL_PARAMETERS=
UPS_SNMP_POLL_NAMES=
UPS_SNMP_POLL_TIMER=
SNMP_COMMUNITY=
SNMP_PORT=
```
